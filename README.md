# Alarm Clock Project

## Overview

This Alarm Clock project is a simple web application that displays the current time and allows users to set alarms. When the current time matches the set alarm, an audio alert is played, and the current time is highlighted in red. The user can also clear the alarm inputs and reset the alarm sound.

## Features

- Displays the current time in a 24-hour format.
- Allows users to set alarms by entering hours and minutes.
- Highlights the current time in red when the alarm goes off.
- Plays an audio alert when the alarm time is reached.
- Provides an option to clear alarm inputs after setting an alarm.
- Manual stop button to turn off the alarm sound.
